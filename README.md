# Prizm-ma 🗺️

Kafe doluluk durumunu gerçek zamanlı olarak harita üzerinde gösteren mobil uygulama.

## Özellikler
- Harita üzerinde kafe konumlarını görüntüleme
- Gerçek zamanlı doluluk takibi
- Konum bazlı filtreleme

## Teknolojiler
- React Native
- Expo
- Firebase Firestore
- Firebase Authentication
- React Native Maps

## Kurulum
```bash
npm install
npx expo start
```
